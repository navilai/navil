# @navil/mcp-security

Agent governance middleware — runtime security proxy for MCP servers.

## Install

Navil is a Python package:

```bash
pip install navil
navil wrap your_mcp_config.json
```

## What it does

Navil sits between your AI agent and MCP servers, detecting and blocking threats in real-time:

- 11 attack patterns detected (prompt injection, data exfiltration, privilege escalation, C2, etc.)
- 368-signature threat blocklist
- Community threat intelligence network
- Policy engine for tool-level access control
- Full audit trail

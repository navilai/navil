CI/CD in minutes

Add MCP security scanning to your GitHub Actions pipeline. Results appear in the Security tab.

GitHub Action

- uses: actions/checkout@v4
- uses: ./.github/actions/navil-scan
  with:
    config: .mcp.json
    fail-on-score-below: "60"
    format: sarif
